## Build Instructions
To build the game, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/NIGHTMARE.git
    cd NIGHTMARE
    ```

2. Compile the game using the Makefile:
    ```bash
    make
    ```

## Usage
After building the game, you can run it with:
    ```bash
    ./NIGHTMARE
    ```
